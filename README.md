Solder-able Breadboard
=======================

This is a bare pcb that is the same size as a mini (35 x 50 mm) breadboard.  It only has the DIP pattern traces, and no power rails.

Repository Contents
-------------------
* **/Hardware** - All Eagle design files (.brd, .sch)
